# Emisor
Este prograba busca ayudar a obtener contenido y automatizar su reproducion usando un strem para poder enviar contenido en tiempo real.
Para hacer funcionar este programa se debe usar los siguientes comandos


git clone https://github.com/SePuedeCambiar/Server.git


cd Server


docker compose up -d --build

Luego de eso deben entrar a la urlde sus dispositivos con el añadido del puerto 9001
En esa pagina se les pedira un admin y contraseña
El admin es 


admin


La contraseña es 


admin_password_2024


El programa requiere de usar una extension de chrome para poder añadir mas sitios en caso de que el usuario lo requiera
https://github.com/SePuedeCambiar/extension
